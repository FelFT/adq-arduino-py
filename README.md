# adq-arduino-py

## Requisitos
- [Tarjeta Arduino UNO]()
- Potenciómetro 
- [Visual Studio Code]()

 ## Conexiones del circuito
Se conecta un potenciómetro a la alimentación de 5 V y a GND. La salida analógica se conecta al pin A0. El circuito se muestra a continuación.
